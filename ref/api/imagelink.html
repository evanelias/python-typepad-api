ImageLink – A link to an image¶

class typepad.api.ImageLink(**kwargs)¶

A link to an image.

Images hosted by TypePad can be resized with image sizing specs. See the url_template field and at_size method.

height¶

The height of the original image in pixels.

If the height of the image is not available (for example, if the image isn’t hosted on TypePad), this property will be omitted.

url¶
The URL for the original, full size version of the image.
url_template¶

An URL template with which to build alternate sizes of this image.

If present, replace the placeholder string {spec} with a valid sizing specifier to generate the URL for an alternate version of this image. This property is omitted if TypePad is unable to provide a scaled version of this image (for example, if the image isn’t hosted on TypePad).

width¶

The width of the original image in pixels.

If the width of the image is not available (for example, if the image isn’t hosted on TypePad), this property will be omitted.
