AuthToken¶

class typepad.api.AuthToken(**kwargs)¶
auth_token¶

The actual auth token string.

Use this as the access token when making an OAuth request.

target_object → TypePadObject¶

Deprecated. The root object to which this auth token grants access.

This is a legacy field maintained for backwards compatibility with older clients, as auth tokens are no longer scoped to specific objects.
