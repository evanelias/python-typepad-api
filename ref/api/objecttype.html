ObjectType¶

class typepad.api.ObjectType(**kwargs)¶
name¶

The name of this object type.

If this is an anonymous type representing the request or response of an action endpoint, this property is omitted.

parent_type¶

The name of the parent type.

This property is omitted if this object type has no parent type.

properties → list of ObjectProperty¶
The properties belonging to objects of this object type.
