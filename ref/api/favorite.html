Favorite – A favorite of some other asset¶

class typepad.api.Favorite(**kwargs)¶

A favorite of some other asset.

Asserts that the user_id and asset_id parameter match ^w+$.

author → User¶

The user who saved this favorite.

That is, this property is the user who saved the target asset as a favorite, not the creator of that asset.

id¶
A URI that serves as a globally unique identifier for the favorite.
in_reply_to → AssetRef¶
A reference to the target asset that has been marked as a favorite.
published¶
The time that the favorite was created, as a W3CDTF timestamp.
url_id¶

A string containing the canonical identifier that can be used to identify this favorite in URLs.

This can be used to recognise where the same favorite is returned in response to different requests, and as a mapping key for an application’s local data store.
