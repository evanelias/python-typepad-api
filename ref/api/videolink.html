VideoLink – A link to a web video¶

class typepad.api.VideoLink(**kwargs)¶

A link to a web video.

embed_code¶
An opaque HTML fragment that, when embedded in a HTML page, provides an inline player for the video.
permalink_url¶

Editable. The permalink URL for the video on its own site.

When posting a new video, send only the permalink_url property; videos on supported sites will be discovered and the embed code generated automatically.
