Post – An entry in a blog¶

class typepad.api.Post(**kwargs)¶

An entry in a blog.

categories → list¶
Editable. A list of categories associated with the post.
embedded_audio_links → list of AudioLink¶
A list of links to the audio streams that are embedded within the content of this post.
embedded_image_links → list of ImageLink¶
A list of links to the images that are embedded within the content of this post.
embedded_video_links → list of VideoLink¶
A list of links to the videos that are embedded within the content of this post.
feedback_status → FeedbackStatus¶
Editable. An object describing the comment and trackback behavior for this post.
filename¶
Editable. The base name of the post to use when creating its permalink_url.
reblog_count¶
The number of times this post has been reblogged by other people.
reblog_of → AssetRef¶
A reference to a post of which this post is a reblog.
